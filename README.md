# Tableau-COVID-Risk-Monitor
A Tableau visualization that surfaces countries with the highest 1-week percentage increase in confirmed cases and the highest Deaths per 100 Cases (Case Fatality Rate). Includes global KPIs for daily changes.

This Tableau dashboard highlights:

# ğŸ§® Key Metrics
# 1. 1-Week % Increase in Confirmed Cases

Identifies countries with the fastest week-over-week growth in infections.

# 2. Deaths per 100 Cases (CFR)

Calculated as:

CFR = (Total Deaths / Total Confirmed Cases) * 100


Highlights countries where COVID-19 outcomes are most severe.

# 3. Global KPIs

Includes:

- Daily new confirmed cases

- Daily new deaths

- % daily change

- Optional 7-day averages
